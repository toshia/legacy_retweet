# Legacy Retweet

## これなに

mikutterに非公式RTを行うためのコマンドを追加します。

mikutter 5.0まであった「引用」機能のバックポートです。引き続き「引用」（非公式リツイート）を使いたい場合、このプラグインをインストールしてください。

## 使い方

インストールして起動すると、mikutterコマンドとして「非公式リツイート」が追加されます。

タイムラインで投稿を右クリックしたときに出るコンテキストメニューに追加されている他、ショートカットキーの設定もできます。

## インストール方法

```sh
mkdir -p ~/.mikutter/plugin && git clone https://github.com/toshia/legacy_retweet.git ~/.mikutter/plugin/legacy_retweet
```

## 動作環境

mikutter 5.1以降で動作します。

5.0以前では、標準機能の「引用」を使ってください。
